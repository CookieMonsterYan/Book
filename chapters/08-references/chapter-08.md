## 8.6 引用格式示例

### 书籍引用
```
[1] 作者. 书名[M]. 版本. 出版地: 出版社, 出版年.
示例：
[1] 吴持恭. 水力学[M]. 第4版. 北京: 高等教育出版社, 2008.
```

### 期刊论文引用
```
[2] 作者. 论文题目[J]. 期刊名, 年, 卷(期): 起止页码.
示例：
[2] Nash J E, Sutcliffe J V. River flow forecasting through conceptual models part I[J]. 
    Journal of Hydrology, 1970, 10(3): 282-290.
```

### 标准引用
```
[3] 标准编号. 标准名称[S]. 发布单位, 发布年.
示例：
[3] GB 50014-2021. 室外排水设计标准[S]. 北京: 中国建筑工业出版社, 2021.
```

### 网络资源引用
```
[4] 作者. 题名[EB/OL]. (发布日期)[引用日期]. 网址.
示例：
[4] Rossman L A. Storm Water Management Model User's Manual Version 5.1[EB/OL]. 
    (2015-06-01)[2024-03-13]. https://www.epa.gov/water-research/storm-water-management-model-swmm.
```
