# 水力模型工程师能力自评问卷

> **用途：** 个人能力评估与发展规划
> **适用对象：** 水力模型工程师、团队管理者
> **评估时间：** 约 30-45 分钟

---

## 使用说明

### 评分标准

每个问题请根据您的实际情况选择最符合的选项：

| 分值 | 等级 | 说明 |
|------|------|------|
| 1 | 从未 | 完全没有相关经验或知识 |
| 2 | 很少 | 偶尔接触，了解基本概念 |
| 3 | 有时 | 有一定经验，能完成基础任务 |
| 4 | 经常 | 经验丰富，能独立处理复杂情况 |
| 5 | 总是 | 精通此领域，能指导他人 |

### 结果解读

完成问卷后，统计各维度得分：
- **总分 85-100：** 建议申请 L5 专家级评估
- **总分 70-84：** 建议申请 L4 高级级评估
- **总分 55-69：** 建议申请 L3 中级评估
- **总分 40-54：** 建议申请 L2 初级评估
- **总分 25-39：** 建议申请 L1 入门评估

## 发展建议

### 根据得分的发展建议

- [ ] 参加水力学/水文学基础培训

- [ ] 考取相关职业资格证书

#### 实践技能 (S) 较弱
- [ ] 增加软件操作练习
- [ ] 参与更多实际项目

#### 项目经验 (E) 不足
- [ ] 主动申请参与新项目
- [ ] 争取承担更多责任
- [ ] 参与不同类型和规模的项目

- [ ] 深入学习公司质量体系
- [ ] 参与质量评审工作
- [ ] 学习质量管理方法论

#### 协作沟通 (C) 需加强
- [ ] 参与技术分享和演讲
