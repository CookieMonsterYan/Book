```python
import pandas as pd

def generate_data_report(data_file):
    """自动生成数据质量报告"""
    df = pd.read_excel(data_file)
    
    report = []
    report.append(f"# 数据质量报告")
    report.append(f"数据文件：{data_file}")
    report.append(f"总记录数：{len(df)}")
    report.append("")
    
    # 检查缺失值
    missing = df.isnull().sum()
    if missing.sum() > 0:
        report.append("## 缺失值检查")
        for col, count in missing[missing > 0].items():
            report.append(f"- {col}: {count}个缺失")
        report.append("")
    
    # 检查异常值（以管径为例）
    if '管径' in df.columns:
        abnormal = df[(df['管径'] < 100) | (df['管径'] > 3000)]
        if len(abnormal) > 0:
            report.append(f"## 异常值检查")
            report.append(f"发现{len(abnormal)}条异常管径记录")
            report.append("")
    
    # 保存报告
    with open('数据质量报告.md', 'w', encoding='utf-8') as f:
        f.write('\n'.join(report))
    
    print("报告已生成：数据质量报告.md")

# 使用
generate_data_report("管网数据.xlsx")
```

### 3.4.1 【核心能力】技术方案设计能力

**【真实案例】如何选择建模方案？**

**场景**：某城市排水规划项目，团队内部对建模方案有分歧：
- 小王：用1D模型就够了，简单高效
- 小李：建议1D-2D耦合，精度更高
- 你作为技术负责人，需要做出决策

**决策框架**：

| 考虑因素 | 1D方案 | 1D-2D耦合方案 |
|---------|-------|--------------|
| 精度要求 | 管网上游够用了 | 需要模拟地面漫流 |
| 时间成本 | 2周 | 4周 |
| 数据基础 | 有完整管网数据 | 还需要地形数据 |
| 客户预算 | 50万 | 80万 |
| 团队能力 | 都能做 | 只有2人会2D |

**决策建议**：
1. 与客户沟通确认：是否需要地面漫流分析？
2. 如果精度要求不高，先做1D，后续可升级
3. 如果要求高精度，申请增加预算和时间

**【实操模板】技术方案对比表**

```markdown
## 项目技术方案对比

### 方案A：1D管网模型
**优点**：
- 建模周期短（2周）
- 团队熟悉，风险低
- 成本可控

**缺点**：
- 无法模拟地面漫流
- 极端暴雨场景精度有限

**适用场景**：常规排水规划、管网改造设计

### 方案B：1D-2D耦合模型
**优点**：
- 精度高，可模拟内涝淹没
- 结果更直观，易获得客户认可

**缺点**：
- 建模周期长（4周）
- 需要高质量地形数据
- 成本增加60%

**适用场景**：内涝风险评估、海绵城市评估

### 推荐方案：方案A + 方案B升级选项
- 一期先用1D完成基础分析
- 如需更高精度，二期升级为1D-2D
```

```python
"""
实验：用机器学习预测模型参数
目标：根据流域特征自动推荐初始参数
"""

import pandas as pd
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import train_test_split

# 1. 准备训练数据
# 收集历史项目的流域特征和最终校准参数
data = pd.read_csv('历史项目参数库.csv')

# 特征：流域面积、平均坡度、不透水率、河网密度等
features = ['area', 'slope', 'impervious', 'drainage_density']
# 目标：糙率系数
 target = 'roughness_coefficient'

X = data[features]
y = data[target]

# 2. 训练模型
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

model = RandomForestRegressor(n_estimators=100)
model.fit(X_train, y_train)

# 3. 评估
score = model.score(X_test, y_test)
print(f"模型R²得分：{score:.3f}")

# 4. 应用：预测新项目的初始参数
new_project = pd.DataFrame({
    'area': [50],  # km²
    'slope': [2.5],  # %
    'impervious': [0.6],  # 60%
    'drainage_density': [1.2]  # km/km²
})

predicted_roughness = model.predict(new_project)
print(f"推荐的糙率系数：{predicted_roughness[0]:.3f}")
```

## 3.7 高效工作习惯：让日常工作更轻松

### 3.7.1 【效率技巧】每日工作法

**晨会15分钟（或与自己对齐）**：
- 今天最重要的3件事是什么？
- 需要谁的支持？
- 预计遇到什么困难？

**深度工作2小时**：
- 关闭微信/钉钉通知
- 专注处理最复杂的任务（如模型调试）
- 使用番茄工作法：25分钟专注 + 5分钟休息

**下班前10分钟**：
- 记录今天完成的事项
- 记录遇到的问题（后续解决后记入知识库）
- 规划明天的工作重点

### 3.7.2 【避坑指南】常见低效陷阱

**陷阱1：完美主义**
- ❌ 总想一次就把模型做到完美
- ✅ 先跑通，再优化，迭代改进

**陷阱2：重复造轮子**
- ❌ 每个项目都从零开始
- ✅ 建立可复用的模板和脚本

**陷阱3：闭门造车**
- ❌ 遇到难题自己死磕一整天
- ✅ 30分钟没思路就求助或查资料

**陷阱4：只做事不总结**
- ❌ 项目做完就忘
- ✅ 每个项目做简单复盘，记录3个收获和改进点
